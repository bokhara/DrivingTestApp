# 题库扩充与更新方案

## 当前状态
- 当前本地题库：600 题
- 数据格式：JSON
- 支持字段：题目、选项、答案、解析、分类、题型、难度

## 目标
逐步扩充到 3000 题左右，并支持：
1. 本地版本号
2. 远程题库版本检查
3. 增量更新 / 全量替换
4. 保留错题与收藏记录

## 推荐数据源策略
### A. 结构化真题源
- 按法规章节整理
- 按题型拆分
- 去重与规范化解析

### B. 生成增强层
- 基于法规与真题风格生成“同知识点变体题”
- 保持 category / difficulty / explanation 字段一致性

### C. 版本化发布
建议服务端提供：
```json
{
  "version": 3,
  "updatedAt": "2026-04-03T22:00:00+08:00",
  "count": 3000,
  "url": "https://example.com/question-bank-v3.json"
}
```

## 本地表字段建议扩展
未来建议新增：
- sourceType: official/generated/manual
- lawReference: 法条引用
- examYear: 年份
- tag: 高速/处罚/灯光/标志/事故处理
- imageUrl: 图片题资源
- isDeleted: 远程下线题目标记

## 扩充路线
### Phase 1
- 600 题 -> 1000 题
- 先补足常见法规、记分、标志、灯光、高速、事故处理

### Phase 2
- 1000 -> 2000 题
- 增加图片区、手势图、标志图、交警手势题

### Phase 3
- 2000 -> 3000+
- 做题目去重、知识点聚类、难度校正、真题映射

## UI 对应建议
- 首页显示：总题数 / 已练 / 错题 / 收藏 / 模拟考试成绩
- 分类页显示各分类题量
- 题目详情页显示法规依据与解析
- 错题本支持“仅看未掌握”
